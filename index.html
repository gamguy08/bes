[Priority 1] MCU 플랫폼 & 통신 개발 경험

1-1. 차량 진단/제어 플랫폼 개발 (2024.04~현재)

프로젝트: 클라우드 진단 플랫폼 & ECOS 전장검사장비 개발

핵심 기술 역량:

  • MCU 플랫폼: STM32H7 시리즈, RTOS 기반 펌웨어 아키텍처 설계
  • 통신 프로토콜 구현: CAN, MQTT, WebSocket, HTTPS 기반 설계

리프로그램/OTA 시스템 설계:

[구현한 OTA 아키텍처] MQTT(명령채널) ←→ STM32H7 ←→ HTTPS(다운로드채널) ↓ ↓ ↓ 상태모니터링 펌웨어 검증 분할전송/재개 에러처리 플래시 Write 무결성 검증 롤백 메커니즘 재부팅 제어 대역폭 관리

1-2. CAN 통신 전문 경험 (2018.10~2020.09)

프로젝트: 농기계 통합 시스템 개발

CAN 통신 핵심 역량:

  • CAN DB 구축: 190종 신호, 41개 CAN ID 정의
  • CANoe 활용: DBC 파일 작성, 실시간 모니터링, 제어기 성능 테스트 자동화
  • CAN 펌웨어 구현: 필터/마스크, 인터럽트 기반 수신, 메시지 큐 관리
  • 통신 게이트웨이 개발: CAN to UDP 변환 보드 개발 (ESP32)

[Priority 2] 리프로그램 & 진단 시스템 개발

2-1. 보안 채널 설계 구현 (2024.04~현재)

  • 인증서 기반 보안 통신: CA/클라이언트 인증서 프로비저닝, Private Key 보안 저장, TLS/SSL 구현
  • 암호화 시스템: WPA2 PSK 환경 PBKDF2 기반 PMK 파생, HSM 연동 (SPI/I2C)
  • 진단 프로토콜 구현: MQTT 기반 실시간 진단 메시지 채널 및 프레임 포맷 설계

2-2. J1772 표준 기반 충전 통신 (2016.12~2019.12)

  • 충전 제어 파일럿 통신 구현: PWM 신호 생성/해석, J1772 State Machine 구현
  • STM32F4 기반 시뮬레이터 개발: 충전기/OBC 모의 장치 및 배터리 프로파일 에뮬레이션
