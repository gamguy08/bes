전기차 충전 통신 시스템

2016.12 ~ 2019.12 STM32F4 · J1772 · PWM

SAE J1772 국제 표준을 구현한 전기차 충전 통신 시스템을 개발했습니다. 충전기(EVSE)와 차량(EV) 간 Pilot 신호 기반 통신 프로토콜을 완전히 구현하고, 양방향 시뮬레이터까지 제작했습니다.

J1772 Pilot 신호 상태 머신 구현

1kHz PWM(±12V) 신호의 전압 레벨로 충전 상태를 제어하고, 듀티 사이클로 최대 공급 전류를 전달하는 양방향 통신 시스템

State A (12V): 차량 미연결 - 대기 상태
State B (9V):  차량 연결 (2.74kΩ 저항) - 충전 준비
State C (6V):  충전 진행 (882Ω 저항) - 전력 공급 중
State D (3V):  환기 요청 (246Ω 저항) - 특수 충전 모드
State E (0V):  전원 차단 - 비상 정지
State F (-12V): EVSE 오류 - 충전 불가

PWM 듀티 사이클 기반 전류 제한 통신

  • 1kHz PWM 신호 생성: ±12V 구형파, TIM2 하드웨어 타이머 활용
  • 듀티 사이클 → 전류 변환 로직:
    • 16% 듀티 → 10A 공급
    • 25% 듀티 → 16A 공급
    • 50% 듀티 → 32A 공급
    • 90% 듀티 → 고속 충전 플래그
  • 전류 계산 알고리즘 구현: 0~850μs 구간 (0.6A/10μs), 850μs 초과 시 ((듀티-640μs)/10μs × 2.5A) 공식 적용
  • 차량 저항 감지: ADC로 Pilot 전압 측정 → 상태 판별 (2.74kΩ/882Ω/246Ω)
  • 상태 천이 로직: 전압 변화 감지 → 릴레이 제어 → 충전 시작/종료

시뮬레이터 개발

  • EVSE 시뮬레이터: 듀티 사이클 가변 PWM 생성, 차량 저항 감지 및 상태 표시
  • OBC 시뮬레이터: 가변 저항 회로로 상태 전환(State B/C), 배터리 충전 프로파일 에뮬레이션
  • 안전 기능: State E 비상 정지, State F 오류 감지, 접지 연속성 확인

연구 성과

  • 논문 2편: 한국자동차공학회 - 전기차 충전 분배 알고리즘, 팬터그래프 충전 테스트 환경
  • 특허 3건: 최대 전력사용 충전 장치, 전기차용 충전장치, 정위치 유도 장치
