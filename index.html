MQTT (Command Channel)          STM32H7          HTTPS (Download Channel)

- Status Monitoring         - Flash Write            - Resume Support
- Error Handling            - Reboot Control         - Checksum Validation

전기차 충전 통신 시스템

2016.12 ~ 2019.12 STM32F4 · J1772 · PWM

J1772 국제 표준을 구현한 전기차 충전 통신 시스템을 개발했습니다. 충전기와 차량 간 PWM 신호 기반 통신 프로토콜을 구현하고 시뮬레이터까지 제작했습니다.

J1772 State Machine 구현

State A: 차량 미연결
State B: 차량 연결, 충전 미준비
State C: 충전 진행
State D: 환기 요청
State E: 단락 감지
State F: 에러 처리

구현 기능

  • PWM 신호 생성/해석 (1kHz, 듀티 10~96%)
  • 전류 제한 정보 듀티 사이클 변조
  • 차량 상태 감지 (저항값 측정)
  • STM32F4 기반 충전기/OBC 시뮬레이터 개발
  • 배터리 충전 프로파일 에뮬레이션

연구 성과
