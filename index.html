차량 진단 & 제어 플랫폼

2024.04 ~ 현재

클라우드로 차량을 진단하고 제어하는 시스템을 만들고 있어요. 멀리 떨어진 곳에서도 차량 상태를 실시간으로 확인하고, 문제를 빠르게 해결할 수 있어요.

무선 펌웨어 업데이트 시스템

MQTT로 명령을 받고, HTTPS로 파일을 다운로드하는 이중 채널 구조예요. 업데이트 중에 문제가 생기면 자동으로 이전 버전으로 돌아가요.

  • 실시간 업데이트 상태 모니터링
  • 파일이 손상되지 않았는지 자동 검증
  • 네트워크가 끊겨도 이어서 다운로드
  • 업데이트 실패 시 자동 롤백

사용한 기술

  • STM32H7으로 고성능 펌웨어 구현
  • FreeRTOS 기반 멀티태스킹 아키텍처
  • TLS/SSL로 보안 통신 구현
  • CAN, MQTT, WebSocket 멀티 프로토콜 지원

농기계 통합 제어 시스템

2018.10 ~ 2020.09

트랙터의 여러 부품들이 서로 통신하며 협력하도록 만들었어요. CAN 통신으로 190개 신호를 실시간으로 주고받아요.

CAN 통신 시스템 구축

190종의 신호와 41개의 CAN ID를 체계적으로 관리하는 데이터베이스를 만들었어요. CANoe로 실시간 모니터링도 가능해요.

  • 190종 신호, 41개 CAN ID 체계 설계
  • DBC 파일 작성 및 CANoe 연동
  • CAN 게이트웨이 개발 (ESP32 기반)
  • 실시간 모니터링 및 테스트 자동화

전기차 충전 시스템

2016.12 ~ 2019.12

전기차를 안전하게 충전하는 시스템을 만들었어요. J1772 국제 표준을 따라서 충전기와 차량이 정확하게 통신해요.

구현한 기능

  • J1772 표준 충전 통신 프로토콜 구현
  • PWM 신호로 충전 상태 제어
  • STM32F4 기반 충전 시뮬레이터 개발
  • 배터리 프로파일 에뮬레이션

연구 성과
