급속충전검사장비 PoC 개발

2025.03 ~ 현재 (PoC 개발) AT32F403A · GBT/CHAdeMO/AC · 타이머 캡처 · 다채널 ADC

급속충전검사장비 PoC 펌웨어 개발 및 장비간 통신 시퀀스 검증을 수행했습니다. Artery AT32F403A 기반으로 GBT/CHAdeMO/AC 멀티 프로토콜 지원 환경에서 CP 신호 측정 정확도를 향상시키고, 통신 안정성을 확보했습니다.

핵심 작업 내용

타이머 캡처 기반 ADC 샘플링 타이밍 최적화: CP 신호(1kHz PWM) 하이 레벨 중간 시점을 타이머 캡처로 감지하고, 해당 이벤트를 ADC 트리거 소스로 설정하여 정확한 전압 측정 구현. 기존 평균 전압 오차 문제 해결.

  • 핀 포팅: AT32F403A 레퍼런스 매뉴얼 분석하여 타이머 캡처 및 ADC 트리거 지원 핀 선정, 하드웨어 제약사항 고려한 핀 매핑 재설계
  • 타이머-ADC 동기화: TIM 캡처 모드로 PWM 상승/하강 엣지 검출 → 하이 레벨 중간 시점 계산 → ADC 트리거 타이밍 동기화
  • 다채널 ADC 구성: CP, PP, 전압, 전류 신호 동시 샘플링 및 DMA 전송
  • CP 전압 레벨 판별: 12V(Idle), 9V(Charging Request), 6V(Charging), 3V(Fault) 상태 정확히 구분
  • 통신 시퀀스 검증: VCP 신호 디바운싱 로직 (3회 연속 검출 + 10초 타임아웃) 구현으로 오검출 방지
  • 타이머 리소스 관리: 500ms 주기 VCP 모니터링 타이머의 생명주기를 통신 세션과 동기화하여 리소스 누수 해결

성과

  • CP 신호 측정 정확도 향상으로 오판정 제로화
  • 멀티 프로토콜(GBT/CHAdeMO/AC) 안정적 지원

LVDS 멀티 ECU 통합 진단 시스템

약 1개월 STM32 · CAN · 11비트/29비트 ID · 동적 필터링

여러 ECU를 통합 진단하기 위해 CAN 표준 ID(11비트)와 확장 ID(29비트)를 동시 처리하는 동적 필터 시스템을 개발했습니다. 기존 시스템은 제어기별로 ID 타입이 고정되어 있어, 확장성을 위한 구조 개선이 필요했습니다.

핵심 작업 내용

  • CAN 필터 동적 설정 구조 설계: 런타임에 11비트/29비트 ID 혼용 처리 가능하도록 필터 뱅크 재설계
  • 설정 데이터 기반 동적 처리: 제어기별 하드코딩 제거, 필터 설정 데이터 수신 → 검증 → 동적 적용 파이프라인 구현
  • 레거시 코드 리팩토링: 예외 처리 남발 방지를 위해 CAN 필터 설정 함수를 ID 타입 매개변수화된 단일 모듈로 통합
  • 시스템 구조 분석: CAN 설정 계층 전체 흐름 (필터링 → 데이터 수신 → 설정 확인) 체계적으로 분석

성과

  • 멀티 ECU 통합 진단 지원 (11비트/29비트 혼용)
  • 차기 개발자를 고려한 유지보수성 확보
